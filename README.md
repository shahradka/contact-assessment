# Contact List app

## How to run

Clone repository by this command

```
https://github.com/shahradka/contact_list

```

*Note: deploy related server repository* 

After that go to project path and use this command for install and deployment

```
npm clean install

```

For debug use:

```
npm start

```


For build use:

```
npm run build

```
